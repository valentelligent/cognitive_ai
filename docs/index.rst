Quick Installation
----------------

.. code-block:: bash

   # Create conda environment
   conda env create -f environment.yml
   
   # Activate environment
   .\activate_env.bat
   
   # Set up development environment
   dev_setup

System Requirements
-----------------

* Windows 10/11
* NVIDIA RTX 2080 SUPER (or compatible)
* CUDA Toolkit 12.1
* Python 3.10
* 8GB+ GPU Memory
